What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from DOM?

To select an element from the DOM in Js. we can use several ways :
1)document.getElementById('id')
2)document.querySelector('selector')
3)document.querySelectorALL('selector')

What is event delegation in the context of Dom and why it is usefull?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for its child elements. It leverages event bubbling to handle events more efficiently, especially for dynamic content, and results in cleaner, more maintainable code.
